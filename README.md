## How to run:

### Frontend:
1. `cd ./frontend`
2. `npm ci`;
3. `npm run start`;

Frontend is accessible on the localhost `http://localhost:1234/`



### Backend:
1. `cd ./backend`
2. `npm ci`;
3. `npm populate`;
4. `npm run serve`;

To clear the database run `npm run clear`. 

Backend is accessible on the localhost `http://localhost:3003/`
